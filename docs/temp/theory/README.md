想着学些数学, 深入地理解一些东西, 却迟迟没有动手. 抱着这样的期待两三年了, 我想简单地学习.

1. 不要花里胡哨地刻意追求自己妄想的, 看看原本知识呈现的
2. 刚开始投入可能难受, 心浮气躁, 看到很多没学, 进度较慢, 也莫心浮气躁
3. 耐心解决问题, 而不是拖沓
