### 感悟

think in the matrix world

就像用 functor 去构建基本操作, 去表达和思考

example:

- $Ax = 0$

  $x$ 有多个解, 这在 matrix world 中就是一个新的矩阵, 即$[x_1, \cdots, x_n] = B$, 于是思考的问题就变成了$AB=0$

  more details: [求解 Ax=0：主变量，特解](https://zhuanlan.zhihu.com/p/45815011)
